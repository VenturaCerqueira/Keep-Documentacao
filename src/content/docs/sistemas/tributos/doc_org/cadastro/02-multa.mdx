## ⚠️ Regras de Negócio Específicas
      <Steps>
      1. Ao cadastrar ou editar uma multa, o campo 'Prazo Final' deve ser igual ou posterior ao 'Prazo Inicial';
      2. Não é possível excluir um cadastro de multa que esteja vinculado a outras operações ou em uso no sistema; 
      </Steps>






## 🧪 Ações / Botões no Sistema:
    <Steps>
        1. **🔍 Pesquisar**: Filtro para pesquisa de todos as multas, pesquisando por "descrição".
            
        2. **👁️‍🗨️ Ações/visualizar**: Acesso para cadastro de visualizar cadastro da multa selecionada.
                ```sh title=" 👁️‍🗨️Ação/visualizar"
                http://www.e-contrib.com.br/gestaotributaria/
                entidade/cadastros/multas/visualizar/1
                ```
        3. **✏️ Editar**: Possibilitando edição dos campos do cadastro de multas
            ```sh title=" 👁️‍🗨️Ação/visualizar"
                http://www.e-contrib.com.br/gestaotributaria/
                entidade/cadastros/multas/criar
            ```
        4. **🗑 Excluir:** Possibilitando excluir multas cadastradas 
        (⚠ Observação: apenas sera possivel excluir caso não possua nenhuma amarração com a multa)
        
   </Steps>
